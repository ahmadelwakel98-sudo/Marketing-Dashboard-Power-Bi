# Marketing-Dashboard-Power-Bi
Marketing Dashboard (Power BI)
Overview
This project focuses on building an interactive marketing dashboard using Power BI. The objective was to analyze marketing performance, track campaign effectiveness, and provide insights that support data-driven marketing decisions.
Data Modeling
• Imported and combined multiple marketing datasets (campaigns, channels, leads)
• Built a structured data model with proper relationships
• Cleaned and transformed data using Power Query
• Optimized the model for better performance and usability
DAX & Calculations
• Created calculated measures using DAX
• Tracked KPIs such as Total Leads, Conversion Rate, Cost per Lead, and ROI
• Built time-based analysis (MTD, growth rate, campaign comparison)
• Applied dynamic calculations based on filters and segments
Dashboard Design
• Designed interactive and user-friendly visuals
• Used charts, cards, and tables to present marketing data clearly
• Added slicers for filtering by campaign, channel, and time
• Focused on clean and professional dashboard layout
Analysis & Insights
• Analyzed campaign performance across different channels
• Identified high-performing and low-performing campaigns
• Evaluated conversion trends and customer behavior
• Provided insights to improve marketing strategy and budget allocation
Outcome
The dashboard delivers a comprehensive view of marketing performance, helping stakeholders track KPIs, optimize campaigns, and make better strategic decisions based on data.
